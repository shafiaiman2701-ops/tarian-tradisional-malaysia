# tarian-tradisional-malaysia
An educational web app introducing Malaysian traditional dances through interactive screens for school projects and exhibitions.

This repository contains a lightweight static web app named "Tarian Tradisional Malaysia" that demonstrates:

- A splash screen with animated title.
- A simple login/language selection screen (Bahasa Melayu / English).
- A main screen listing traditional dances as large buttons.
- A detail screen showing origin, community, function, and a short description of the dance.

Run locally
-----------

Open `index.html` directly in a browser, or serve the folder with a simple HTTP server:

```bash
# from the repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Files
-----

- `index.html` : App shell and views.
- `styles.css` : Styling for screens and animations.
- `app.js` : App logic, data for dances, and view navigation.

Feel free to request enhancements (audio, video, or richer animations).
